# Solusi Scene Hilang Saat Play

## ❌ MASALAH: Scene hilang atau tidak muncul saat play game

## ✅ SOLUSI:

### Langkah 1: Setup Build Settings
1. Buka menu: **File > Build Settings**
2. Di bagian "Scenes In Build", pastikan kedua scene sudah ada:
   - **Scene 0**: MainMenu (drag Assets/Scenes/MainMenu.unity ke sini)
   - **Scene 1**: Gameplay (drag Assets/Scenes/Gameplay.unity ke sini)

**PENTING:** Urutannya harus MainMenu di index 0, Gameplay di index 1

### Langkah 2: Setup SceneManager di MainMenu
1. Buka scene "MainMenu"
2. Di Hierarchy, cari atau buat GameObject "SceneManager"
3. Attach Component: `SceneManager` script
4. **Optional:** Di Inspector, bisa langsung assign:
   - **Play Button**: Drag tombol play dari hierarchy
   - **Quit Button**: Drag tombol quit dari hierarchy
   
   (Jika tidak di-assign, script akan auto-find berdasarkan nama)

### Langkah 3: Setup Tombol dengan Nama Benar
Pastikan tombol di scene MainMenu namanya:
- **"PlayButton"** (case-sensitive)
- **"QuitButton"** (case-sensitive)

Atau assign manual di SceneManager Inspector field.

### Langkah 4: Save Scene
**PENTING:** Sebelum play, pastikan scene sudah di-save dengan `Ctrl+S`

### Langkah 5: Test Play
Sekarang klik Play → klik tombol Play di menu → harusnya langsung ke Gameplay scene

## 🔍 Debugging:

Jika masih hilang, buka **Console** (Ctrl+Shift+C) dan cek:
- Error apa yang muncul?
- Apakah ada pesan "[SCENE] Loading Gameplay..."?

## Checklist:
✅ Build Settings sudah punya MainMenu (index 0) dan Gameplay (index 1)
✅ SceneManager attach ke GameObject di MainMenu
✅ Tombol di-name dengan benar (PlayButton, QuitButton) ATAU di-assign di Inspector
✅ Scene sudah di-save sebelum play
✅ Tidak ada error di Console

Jika masih ada masalah, cek Console untuk error message!
